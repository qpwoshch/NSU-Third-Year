# Места (Асинхронное сетевое взаимодействие)

Используя методы асинхронного программирования (CompletableFuture), провзаимодействовано с несколькими публично доступными API и сделано приложение с интерфейсом, основанное на этих API.  

## Логика работы:

1. Пользователь вводит название чего-то (например "Цветной проезд") и нажимает кнопку поиска.
2. Ищутся варианты локаций с помощью метода [1] и показываются пользователю в виде списка.
3. Пользователь выбирает одну локацию.
4. Ищется погода в локации.
5. Ищутся интересные места в локации, далее для каждого найденного места с помощью метода [1] ищутся описания.
6. Всё найденное показывается пользователю.

## Для запуска:

При запуске через IntelliJ IDEA нужно в параметрах запуска в VM Options указать путь до JavaFX.
Создайте файл `.env` и добавьте туда API ключи для GraphHopper и OpenWeatherMap.
